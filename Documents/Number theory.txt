1: Công thức legendre:
+ v_p(n!): Số nguyên k lớn nhất thỏa mãn n! chia hết p^k, số nguyên dương n và số nguyên tố p cho trước 
v_p(n!) = [n / p] + [n / p^2] + ...... với p^i <= n
+ Đối với p là hợp số (p không phải số nguyên tố): p = p1^r1 * p2^r2 *....
ans = min(v_pi(n!) / r_i) với 1 <= i <= n 
________________________________________________________________________________________________________________________________

2: Phi hàm euler: 
Phi(N): Số nguyên tố cùng nhau với N trong đoạn [1, N]
Phi(n) = n * (1 - 1 / p) với mọi p là các ước nguyên tố của N
       = n (1-1/p1) ... (1-1/pn)
________________________________________________________________________________________________________________________________

3: Các cách tính số ước
d(n): Số ước của n
+ d(n^2) thỏa mãn < n = d(n^2) / 2
+ n = e1^p1 * e2^p2 * .....
       d(n) = (p1 + 1) * (p2 + 1)......
       d(n^2) = (2*p1 + 1) * (2*p2 + 1).....
+ Tìm d(n!):
       B1: Sáng snt từ 1 -> n
       B2: với mỗi snt p <= n, tìm v_p(n!) chính là số mũ của snt p trong phân tích n!
       d(n^!) = (v_p1(n!) + 1)(v_p2(n!) + 1).... với mọi snt p <= n
________________________________________________________________________________________________________________________________

4: Phương trình
+ Đếm số cặp nghiệm của phương trình: 1/x + 1/y = 1/n 
       1/x + 1/y = 1/n  <=> (y - n)(x - n) = n^2
       ycbt <=> số nghiệm = d(n^2)
________________________________________________________________________________________________________________________________

5: 1 số đẳng thức lưu ý
+ 1^2 + 2^2 + 3^2 + ... + n^2 = n(n + 1)(2n + 1) / 6
+ 1^3 + 2^3 + 3^3 + ... + n^3 = (1 + 2 + ... + n)^2
+ 2 + 5 + 8 + ... + (3n - 1) = n(3n + 1) / 2
+ 1 + 3 + 5 + ... + (2n - 1) = n^2
________________________________________________________________________________________________________________________________

6: 1 số định lý
+ Định lý fermat nhỏ: a^(p - 2) đồng dư a^-1 (mod p) với p là snt, a không chia hết cho p
